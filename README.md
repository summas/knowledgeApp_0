# 機能一覧
- 記事投稿機能(Action Text)
- 記事詳細表示機能
- 記事一覧表示機能（React)
- カテゴリーによる条件抽出（React)
- ページネーション(React, Material-ui)
- ページネーション(Kaminari)
- ユーザー登録機能(Device)
- 画像ファイルのアップロード機能(CarrierWave)
- 簡単ログイン機能
- テスト機能(RSpec)

#  使用技術
#### サーバーサイド
- Ruby 2.6.3
- Rails 6.0.2.2
- Nginx
- Puma
- SSL (Let's Encrypt)

#### フロントエンド
- React 16.13.1, Typescript 3.9.5
- Bootstrap 4
- Material-UI

#### AWS
- VPC
- RDS(Mysql 5.7.26)
- EC2
- S3
- Route53
